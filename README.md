## Development

Clone the repository and install the dependencies:

```sh
git clone https://github.com/xbmlz/starter-gin.git

cd starter-gin

go mod download
```

Run the application:

```sh
make run
```

## Build

Build the application:

```sh
make build
```

```sh
make docker
```
